# Agent instructions for vibe2ship

## What this is

A hackathon/competition starter repo. The only file is a problem statement: build a **Community Hero — Hyperlocal Problem Solver** platform.

**No code, no config, no build system, no git history exists yet.** Everything must be created from scratch.

## The challenge

A platform for citizens to report, track, and resolve community issues (potholes, water leaks, streetlights, waste, etc.) using AI.

Core feature areas (from the problem statement):
- Image/video-based reporting
- AI-powered issue categorization
- Geo-location and mapping
- Community verification
- Real-time tracking
- Impact dashboards
- Predictive insights
- Gamification
